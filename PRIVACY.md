## English Version 

### Overview 

The File Converter Plugin ("Plugin") is designed to facilitate local file conversions between various formats while prioritizing user privacy and data security. This privacy policy explains how we handle your data when you use our plugin within the Dify platform. 

### Data Collection 

**What we DO NOT collect:** 
- Personal identification information 
- User account details 
- Usage analytics or tracking data 
- Device information 
- Location data 
- Cookies or similar tracking technologies 

**What we process:** 
- File contents you choose to convert 
- File metadata (e.g., filename, content type) required for conversion operations 
- Temporary processing data required for file operations 

### Data Processing 

**Local Processing:** 
- All file conversions are performed locally on your system without external data transmission 
- No files are uploaded to external servers during the conversion process 
- Temporary files are created and deleted securely during processing 

### Data Storage 

**No Data Retention:** 
- We do not store or retain copies of your files after conversion 
- We do not access or view the contents of your files beyond what is necessary for conversion 
- Temporary files are automatically deleted after the conversion process completes 

### Security Measures 

**Data Protection:** 
- All file processing is performed in a secure local environment 
- No network connections are made for file conversion operations 
- Temporary files are handled with appropriate security measures

## 中文版本 

### 概述 

文件转换器插件（"插件"）旨在促进各种格式之间的本地文件转换，同时优先保护用户隐私和数据安全。本隐私政策说明了您在Dify平台内使用我们的插件时，我们如何处理您的数据。 

### 数据收集 

**我们不收集的内容：** 
- 个人身份信息 
- 用户账户详细信息 
- 使用分析或跟踪数据 
- 设备信息 
- 位置数据 
- Cookie或类似跟踪技术 

**我们处理的内容：** 
- 您选择转换的文件内容 
- 转换操作所需的文件元数据（如文件名、内容类型） 
- 文件操作所需的临时处理数据 

### 数据处理 

**本地处理：** 
- 所有文件转换都在您的系统上本地执行，无需外部数据传输 
- 转换过程中不会将文件上传到外部服务器 
- 处理过程中会安全地创建和删除临时文件 

### 数据存储 

**不保留数据：** 
- 转换完成后，我们不存储或保留您文件的副本 
- 除了转换所需的内容外，我们不访问或查看您文件的内容 
- 转换过程完成后，临时文件会自动删除 

### 安全措施 

**数据保护：** 
- 所有文件处理都在安全的本地环境中执行 
- 文件转换操作不会建立网络连接 
- 临时文件通过适当的安全措施进行处理
